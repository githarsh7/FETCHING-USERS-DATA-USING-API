# Fetch Users Data Using API

A simple JavaScript project that fetches users data from an API using the Fetch API with both Promise-based (`.then()` / `.catch()`) and `async/await` methods.

The project displays user information on the webpage and also logs the API response data in the browser console.

---

## Features

- Fetch data from a public API
- Use Fetch API in JavaScript
- Handle API responses properly
- Convert response to JSON format
- Display users data on the webpage
- Log users data in the browser console
- Error handling using `.catch()` and `try...catch`
- Implement both:
  - Promise-based approach
  - Async/Await approach
